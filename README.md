# Reversi Moji

This project is a revisited Reversi game

## Installation

```bash
npm install
```

## Run in development

```bash
npm run dev
````

Open http://localhost:3000

## Run in production

```bash
npm run build
npm run start
```

## Run linter
```bash
npm run lint
```
